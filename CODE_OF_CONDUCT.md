# Código de Conduta

Como participantes deste projeto, nos comprometemos a respeitar todas as pessoas que contribuem, reportam bugs, postam feature requests, atualizam documentação ou qualquer outra atividade.

- Seja respeitoso e acolhedor.
- Não serão tolerados comportamentos abusivos, discriminatórios ou ofensivos.
- Reporte violações para security@agrotm.com.

Inspirado pelo [Contributor Covenant](https://www.contributor-covenant.org/).
