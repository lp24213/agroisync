# Deploy AGROTM

## 1. GitHub

- Crie um repositório: agrotm-site
- Faça push do projeto completo

## 2. Vercel

- Acesse https://vercel.com
- Clique em "New Project" e importe o repositório GitHub
- Configure:
  - Framework: Next.js
  - Root directory: /
  - Variables de ambiente: copie de .env.example

✅ Vercel cuidará do deploy contínuo sempre que fizer push no GitHub.
