# Scripts AGROTM

## install.sh
Instala todas as dependências do monorepo (pnpm, backend, frontend, contratos).

## dev.sh
Roda o ambiente de desenvolvimento de todos os workspaces.

## deploy.sh
Faz deploy do backend (Firebase), frontend (Vercel) e orienta deploy dos contratos.
