# AGROTM DeFi Dashboard

Dashboard DeFi para a plataforma AGROTM Solana.

## Funcionalidades

- Visualização de pools de liquidez
- Análise de yield farming
- Monitoramento de portfólio
- Análise de risco
- Relatórios de performance

## Tecnologias

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Solana Web3.js

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```
