# AGROTM – Plataforma Web3 del Agro (ES)

AGROTM es el futuro descentralizado del agronegocio. Un ecosistema con NFTs, staking, panel de inversores y alta seguridad.

## Funcionalidades

- Mint de NFT vía IPFS
- Staking multichain
- Panel administrativo + whitelist
- Gráfico de token en vivo

## Ejecutar

```bash
npm install
npm run dev
```
