# AGROTM – Plataforma Web3 do Agro (PT)

O projeto AGROTM é a revolução do agronegócio via blockchain. Um ecossistema descentralizado com NFT, staking, painel de investidores e segurança avançada.

## Funcionalidades
- Mint de NFT via IPFS
- Dashboard de staking multichain
- Painel de administração e whitelist
- Tokenomics com gráfico on-chain

## Como rodar
```bash
npm install
npm run dev
```
