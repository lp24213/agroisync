# Playbook: Resposta a Vazamento/Roubo de Dados

## 1. Detecção
- Monitoramento detecta acesso/suspeita de vazamento (alertas, logs, SIEM)
- Confirmar incidente via análise de logs, dashboards, alertas

## 2. Contenção
- Isolar sistemas afetados
- Revogar acessos comprometidos
- Ativar rotação de segredos e credenciais

## 3. Comunicação
- Notificar time de segurança, DPO e stakeholders
- Atualizar status page e, se necessário, órgãos reguladores

## 4. Erradicação
- Identificar causa raiz e vetor de ataque
- Corrigir vulnerabilidades
- Atualizar sistemas e segredos

## 5. Recuperação
- Restaurar sistemas a partir de backup
- Validar integridade dos dados
- Monitorar acessos e estabilidade

## 6. Pós-incidente
- Documentar incidente, resposta e lições aprendidas
- Atualizar playbooks, políticas e treinamentos

## Checklist Rápido
- [ ] Incidente detectado e confirmado
- [ ] Contenção aplicada
- [ ] Time de segurança e DPO notificados
- [ ] Stakeholders e órgãos reguladores informados
- [ ] Causa raiz identificada
- [ ] Recuperação executada
- [ ] Pós-incidente documentado
