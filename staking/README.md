# AGROTM Staking

Módulo de staking da plataforma AGROTM Solana.

## Funcionalidades

- Staking de tokens AGRO
- Unstaking de tokens
- Reivindicação de recompensas
- Análise de APR
- Histórico de transações

## Tecnologias

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Solana Web3.js

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```
