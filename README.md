# AGROTM Solana

Plataforma DeFi para o setor agroindustrial usando Solana.

## Instalação

```sh
npm install
```

## Desenvolvimento

```sh
npm run dev
```

## Build de produção

```sh
npm run build
npm start
```

## Deploy

Deploy automático na Vercel.

## Variáveis de ambiente

Configure o arquivo `.env` com as chaves reais do seu ambiente.
